# Tokex
This is a repository for assorted files based on the Toki Pona dialect/script I made, which uses hexadecimal digits to spell out words.

![image](https://github.com/AbbyRead/Tokex/blob/c15dd30cc2e2f21eb22cd17ee8af7b8fdc93fada/Tokex%20in%20Minicube64.png)
| Toki Pona<br>Latin Phoneme | Tokex<br>Equivalent |
|---:|:---|
| A / a | `A` |
| I / i | `1` |
| U / u | `2` |
| E / e | `E` |
| O / o | `6` |
| P / p | `B` |
| T / t | `D` |
| K / k | `C` |
| M / m | `8` |
| N / n | `9` |
| L / l | `7` |
| S / s | `5` |
| J / j | `4` |
| W / w | `3` |
| Space | `00` |
| ". " | `0000` |
| Pad | `6` |
